## 코딩실습 1: 육면체에 사진을 입히고 회전하기
- GPT를 활용한 코딩

### 프람프트:
```
P5.js 로 
큐브를 하나 만들고, 
img.png 로 택스쳐 매핑해 줘
orbitControl 로 제어 가능하게 해 줘
```

```
* p5.js로 **큐브 생성**
* `img.png`를 **텍스처로 매핑**
* `orbitControl()`을 이용해 **마우스로 회전/확대/이동** 가능
```

---

### 🧱 1. 3D 모드로 canvas 만들기

* `createCanvas(width, height, WEBGL)`을 사용해 **3D 렌더링 모드**로 캔버스를 생성해요.
* `WEBGL` 모드를 통해 p5.js가 3D 공간을 이해하게 만들어요.

---

### 🖱️ 2. orbitControl로 마우스 제어

* `orbitControl()` 함수를 `draw()` 안에 넣으면,

  * **마우스로 드래그**해서 회전
  * **마우스 휠**로 줌 인/아웃이 가능해져요.
* 카메라를 직접 조작하지 않아도 쉽게 3D 조작을 구현할 수 있어요.

---

### 🖼️ 3. 텍스처 이미지 불러오기

* `loadImage('img.png')`로 이미지를 불러와요.
* 이 이미지는 전역 변수에 저장해서 사용해요.
* 이 과정은 `preload()` 함수 안에서 진행하면 렌더링 전에 미리 로딩할 수 있어서 깔끔해요.

---

### 🧊 4. 큐브에 텍스처 매핑하기

* `texture(img)` 함수를 큐브 앞에 사용해서 텍스처를 입혀요.
* 그리고 `box(size)`를 사용해 정육면체 큐브를 그려요.
* p5.js는 기본적으로 큐브 6면 전체에 동일한 텍스처를 매핑해 줘요.

---

### 🎯 5. 전체 흐름 요약

1. `preload()`에서 `img.png` 이미지 로딩
2. `setup()`에서 3D canvas 생성
3. `draw()`에서는

   * `background()`로 배경 설정
   * `orbitControl()`로 마우스 제어
   * `texture(img)`로 텍스처 설정
   * `box(크기)`로 큐브 렌더링

## 코딩실습 2

### 프람프트
- **p5.js로 블록깨기 게임을 만들 때**의 전체 구성과 
- 주요 로직을 **PPT 5장 분량**에 맞춰 
- **슬라이드별 설명 형식**으로 정리 해줘

---

### 🎯 **슬라이드 1: 게임 개요**

* **게임 이름**: 블록깨기 (Breakout Game)
* **사용 도구**: p5.js (웹 브라우저에서 실행되는 자바스크립트 그래픽 라이브러리)
* **게임 목표**: 공을 튕겨서 위쪽에 있는 블록들을 모두 없애면 승리
* **주요 요소**:

  * 공 (ball)
  * 패들 (paddle, 바닥에서 움직이는 판)
  * 블록들 (blocks)
  * 충돌 판정 (collision detection)

---

### 🎮 **슬라이드 2: 게임 화면 구성**

* **배경(canvas)**: 2D 게임 화면
* **공(ball)**: 원(circle), 자동으로 움직이며 패들/벽에 튕김
* **패들(paddle)**: 사용자가 마우스 또는 키보드로 좌우 조작
* **블록(blocks)**: 위쪽에 정렬되어 있으며 공과 충돌하면 사라짐
* **구성 방법**:

  * `setup()`에서 화면 크기 지정
  * `draw()` 반복문에서 매 프레임마다 화면 업데이트

---

### 🧠 **슬라이드 3: 게임 로직 흐름**

* **1. 공의 움직임**

  * `x`와 `y` 좌표를 `speedX`, `speedY`로 이동
  * 벽이나 패들, 블록에 닿으면 반사됨

* **2. 충돌 판정**

  * 공이 화면 벽, 패들, 블록에 닿는지 계산
  * `if` 조건문으로 충돌 시 방향 전환 또는 블록 제거

* **3. 게임 상태**

  * 블록이 모두 사라지면 "승리" 메시지 출력
  * 공이 아래로 떨어지면 "게임 오버"

---

### 🕹️ **슬라이드 4: 사용자 입력과 반응**

* **입력 방식**:

  * 마우스 움직임 → 패들이 따라옴
  * 또는 키보드 방향키로 좌우 조작

* **패들 조작 방식**:

  * `mouseX` 값을 사용해 패들의 `x` 위치 지정
  * 또는 `keyIsDown(LEFT_ARROW)` / `keyIsDown(RIGHT_ARROW)`로 제어

* **난이도 조절**:

  * 공의 속도, 패들 크기, 블록 수로 난이도 조절 가능

---

### 🌈 **슬라이드 5: 게임 확장 아이디어**

* **디자인 향상**

  * 무지개 색 블록 적용 (`fill()`에 색상 배열 사용)
  * 배경음악과 효과음 추가

* **레벨 시스템**

  * 블록이 더 많거나 속도가 빨라지는 레벨 구성

* **점수 시스템**

  * 블록을 깰 때 점수 증가
  * 화면 상단에 점수 표시

* **모바일 대응**

  * 터치로 패들 조작 가능하게 확장
