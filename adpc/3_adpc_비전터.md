# ✅ **3장. 실습 데모 ① 음성인식 (1분)**

# ✅ **4장. 실습 데모 ② 카메라 인식 (1분)**

* 얼굴/사물을 비추면 AI가 자동 분류
* AI의 “이해”는 **진짜 이해가 아니라 확률 계산**
  → 인간의 마음·양심·사랑과는 다름

# ✅ **5장. AI의 짧은 역사 (1분)**

* 1950: 튜링 테스트
* 1990s: 패턴인식·영상처리 (교수님 연구 분야)
* 2012: 딥러닝 혁명
* 2023~25: ChatGPT 등 LLM 시대
  → **AI는 갑자기 나온 게 아니라, 쌓이고 또 쌓인 지혜의 역사**
  (안동교회 110년 역사처럼!)

---

# ✅ **6장. LLM이 뭔가요? (1분)**

* LLM = **말을 잘하는 인공지능**
* 할 수 있음: 글쓰기, 요약, 수학 풀이, 성경배경 설명
* 할 수 없음: **감정, 양심, 영적 판단**
  → ChatGPT = “언어 계산기”지, 사람은 아님

---

# ✅ **7장. AI 시대 진로 팁 (2분)**

### ✨ 유망 분야

* AI 엔지니어
* 컴퓨터비전·드론·로봇
* 반도체·데이터 분석

### ✨ 필요한 능력

* 프로그래밍(C·Python)

* 문제 해결 + 꾸준함
  → “누구나 시작할 수 있다”가 AI 시대의 장점!

# ✅ **8장. 학생들이 준비해야 할 3가지 (1분)**

1. **기본기**: 수학 + 영어
2. **디지털 문해력**: 파이썬, AI 활용법
3. **인성과 정직**: AI가 절대 대체 못하는 영역
   → AI는 똑똑, 사람은 **양심**이 있다!

# ✅ **9장. AI 시대의 믿음 (2분)**

* 가짜 뉴스·딥페이크가 넘치는 시대
  → **말씀과 성령의 분별력** 필수
* AI는 거짓을 말할 수 있지만
  **하나님의 말씀은 진리**
* “너희는 세상의 빛과 소금”(마 5:13~16)
  → AI는 도구,
  우리의 목적은 언제나 **하나님께 영광**

# ✅ **10장. 안동교회의 정체성과 우리의 사명 (2분)**

* 안동교회 석조 예배당 = 등록문화재 **654호**
* 1909년부터 지역을 밝힌 **어머니 교회**
* 표어: **Soli Deo Gloria**
* 기술은 변해도 우리의 정체성은 변하지 않음
  → AI 시대 청소년들의 사명:
  **지혜롭게 배우고, 선하게 사용하고, 하나님께 영광!**
